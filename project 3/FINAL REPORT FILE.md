**### Introduction**

This project analyzes employee attrition data using Python, Power BI, and Sklearn. It integrates data cleaning, visualization, and predictive modeling to provide actionable insights for HR teams.



**### Abstract**

The workflow combines Pandas and Seaborn for preprocessing and exploration, Power BI for interactive dashboards, and Sklearn for machine learning. The objective is to identify attrition drivers and predict employee risk.



**### Tools Used**

\- \*\*Python (Pandas, Seaborn):\*\* Data cleaning and visualization  

\- \*\*Power BI:\*\* Interactive dashboards and reporting  

\- \*\*Sklearn:\*\* Predictive modeling and classification  



**### Steps Involved in Building the Project**

1\. Load and clean datasets with Pandas  

2\. Explore and visualize patterns using Seaborn  

3\. Build interactive dashboards in Power BI  

4\. Train and evaluate predictive models with Sklearn  

5\. Integrate findings into a unified report  



**### Conclusion**

By combining Python, Power BI, and Sklearn, the project delivers a comprehensive view of attrition. It highlights key trends, predicts employee risk, and supports HR in making data‑driven retention strategies.
